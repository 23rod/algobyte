Tipos de Datos

En Python, los tipos de datos son fundamentales para definir el tipo de información que una variable puede almacenar. Conocer los tipos de datos y cómo utilizarlos correctamente es esencial para escribir programas eficaces y claros.

Tipos de Datos Básicos

Python soporta varios tipos de datos básicos que se pueden dividir en las siguientes categorías:

Tipos Numéricos Enteros

int: Números enteros. Ejemplo: edad = 25

Tipos Numéricos de Punto Flotante

float: Números decimales de precisión simple. Ejemplo: altura = 1.75

complex: Números complejos, que tienen una parte real y una imaginaria. Ejemplo: numero = 3 + 4j

Tipos de Secuencias

str: Cadenas de caracteres (texto). Ejemplo: nombre = 'Juan'

list: Listas que pueden contener múltiples elementos de diferentes tipos. Ejemplo: numeros = [1, 2, 3]

tuple: Tuplas, similares a las listas pero inmutables. Ejemplo: coordenadas = (10, 20)

Tipos Booleanos

bool: Valores booleanos (True o False). Ejemplo: es_adulto = True

Tipos de Conjunto y Diccionario

set: Conjuntos, colecciones no ordenadas de elementos únicos. Ejemplo: conjunto = {1, 2, 3}

dict: Diccionarios, colecciones de pares clave-valor. Ejemplo: persona = {'nombre': 'Ana', 'edad': 30}

En resumen, Python ofrece una variedad de tipos de datos básicos que se adaptan a diferentes necesidades. Conocer cómo usarlos te permitirá manipular y almacenar datos de manera eficiente en tus programas.

Tipos de Datos

En C++, los tipos de datos son esenciales para definir el tipo de información que una variable puede almacenar. Conocer los tipos de datos y cómo utilizarlos correctamente es fundamental para escribir programas robustos y eficientes.

Tipos de Datos Básicos

C++ soporta varios tipos de datos básicos que se pueden dividir en las siguientes categorías:

Tipos Numéricos Enteros

int: Números enteros. Ejemplo: int edad = 25;

short: Enteros cortos. Ejemplo: short cantidad = 100;

long: Enteros largos. Ejemplo: long distancia = 100000L;

unsigned: Enteros sin signo (no negativos). Ejemplo: unsigned int poblacion = 3000000;

Tipos Numéricos de Punto Flotante

float: Números decimales de precisión simple. Ejemplo: float altura = 1.75f;

double: Números decimales de doble precisión. Ejemplo: double peso = 70.5;

long double: Números decimales de precisión extendida. Ejemplo: long double pi = 3.141592653589793238L;

Tipos de Caracteres

char: Caracteres individuales. Ejemplo: char inicial = 'A';

wchar_t: Caracteres anchos. Ejemplo: wchar_t letra = L'Á';

Tipos Booleanos

bool: Valores booleanos (true o false). Ejemplo: bool esAdulto = true;

Tipos Vacíos

void: Representa la ausencia de tipo. Usado principalmente en funciones.

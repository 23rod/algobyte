Tipos de Datos

Las estructuras de control condicional en C++ permiten que el flujo del programa se desvíe en función de ciertas condiciones. Las más comunes son if, else if, else y switch. Estas estructuras son esenciales para la toma de decisiones en los programas.

Estructura if
La estructura if evalúa una condición y ejecuta un bloque de código solo si la condición es verdadera.

Estructura if-else
La estructura if-else proporciona una alternativa cuando la condición es falsa.

Estructura if-else if-else
La estructura if-else if-else permite evaluar múltiples condiciones en secuencia.

Estructura switch
La estructura switch evalúa una expresión y ejecuta el bloque de código correspondiente a su valor.

Operadores Relacionales y Lógicos
Para construir condiciones, se utilizan operadores relacionales y lógicos.

Operadores Relacionales:

==: Igual a
!=: Diferente de
<: Menor que
<=: Menor o igual que
>: Mayor que
>=: Mayor o igual que

Operadores Lógicos:

&&: AND lógico
||: OR lógico
!: NOT lógico
